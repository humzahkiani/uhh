# User Flows

## Core
1. As a dev working in the terminal, when I forget a specific command I want to use I want to be able to enter a phrase that describes what the command does and have the commands that match the phrase returned and ranked by how well they match. 

## Future

1. As a dev working in the terminal, when uhh does not match a phrase to a command, I want to be able to add it as a phrase for an existing command or create a new command.
