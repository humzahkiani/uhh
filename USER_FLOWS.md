# User Flows

## Core
1. As a dev working, when I forget a specific terminal command I want to use I want to be able to enter a phrase that describes what the command does and have the commands that match the phrase returned and ranked by how well they match
2. As a dev working in the terminal, I want to be able to save new commands so that I can search for them when I forget them

## Future

1. As a dev working in the terminal, when uhh does not match my phrase to the correct command but the command has been saved, I want to be able to add my phrase to the saved command so future match's will retrieve it
2. As a dev, I want to be able to designate placeholder sections in the commands that I save for the arguments of the command
